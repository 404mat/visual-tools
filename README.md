## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)

### Installation

```bash
git clone https://github.com/404mat/visual-tools.git
cd visual-tools
pnpm install
```

### Running the App

```bash
pnpm run dev
```

The app will be available at [http://localhost:5173](http://localhost:5173).
